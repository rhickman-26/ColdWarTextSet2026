# ColdWarTextSet2026
An interactive digital text set for high school U.S. History, covering the Cold War from 1945-1991.

Overview
This resource was created for Burke County High School U.S. History teachers to address ongoing student struggles with understanding the Cold War. It includes 13 digital primary sources organized into three galleries, aligned to Georgia Standards SSUSH20-23.

Features
Three chronological galleries (Truman/Eisenhower, Kennedy/Johnson, Nixon through Reagan)
Learning targets and success criteria with self-assessment checkboxes
Progress tracking that saves between sessions
Guiding questions for each text
Synthesis questions for each gallery
Enrichment book recommendations
